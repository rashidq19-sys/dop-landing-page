New Features Section Layout

4 features: Live Tracking, Same Day Delivery, Arriving, Automatic Data Backup. Which layout style?

A

Alternating rows — text + screenshot

Matches the rest of the page exactly. Each feature gets its own full-width row — copy on one side, screenshot placeholder on the other, alternating left/right. Scroll animations, same "browser frame" treatment as other sections.

← [Live Tracking copy]         [screenshot →]
→ [screenshot]            [Same Day copy →]
← [Arriving copy]          [screenshot →]
→ [screenshot]            [Data Backup copy →]
+ Consistent with the rest of the page  ·  Longer scroll
B

2×2 card grid — icon + copy only

Four cards in a clean 2-column grid. Each card has an icon, feature name, and 2–3 lines of copy. No screenshots needed — compact and scannable. Works well as a "What's New" block without breaking the page flow.
